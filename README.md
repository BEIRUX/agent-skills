# Agent Skills by BEIRUX

Free, open-source skills for AI coding agents. One command install.

Works with **Claude Code**, **Cursor**, **Gemini CLI**, and **Codex CLI**.

## Install

```bash
# Install a specific skill
npx playbooks add skill BEIRUX/agent-skills --skill <skill-name>

# Or clone and copy manually
git clone https://github.com/BEIRUX/agent-skills.git
```
